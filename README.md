## 📌 Overview

Call Center Manage Service assists CC/CS agents in performing daily outreach to customers or merchants. Tasks are automatically assigned, and agents are guided through the conversation process with:

- 📋 **Ticket information** — full customer ticket details displayed in the interface
- 💬 **Conversation scripts** — prepared slides that guide agents during calls
- ✅ **Action confirmation** — agents confirm results and determine next steps after each interaction

## ⚙️ Key Features

| | Feature | Description |
|:---:|---|---|
| 🧾 | **Automatic Ticket Assignment** | Customer tickets are automatically distributed to agents |
| 💬 | **WhatsApp Desktop Integration** | Instantly open WhatsApp Desktop to chat or call customers |
| 🖥️ | **Guided Conversation Slides** | Prepared scripts guide agents through each interaction |
| 📋 | **Ticket Information Display** | Full ticket details visible so agents understand the issue |
| ✅ | **Action Confirmation** | Agents confirm outcomes and set next actions after conversations |

## 🎯 Purpose

Help Call Center teams perform structured and consistent customer follow-ups:

- ✔️ Customers are contacted daily
- ✔️ Agents follow standardized conversation scripts
- ✔️ Ticket handling becomes faster and more organized
